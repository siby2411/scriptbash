# Scripts de Déploiement Auto-Pro & ERPs

Ce dépôt contient les scripts d'automatisation pour le lancement de vos applications.

## Scripts disponibles
- `deploy_services.sh` : Lance les applications ERP de base.
- `deploy_services_cloudflared.sh` : Lance les applications avec tunnels Cloudflare.
- `juin1.sh` : Script maître de déploiement massif. Gère le lancement simultané de plus de 20 applications depuis le répertoire /root/apachewsl2026/.
